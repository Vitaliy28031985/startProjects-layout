# Finance Ledger


Для встановлення введіть в терміналі: npm install

## Команди

### Запуск сервера для разробки
```shell
npm run start
```

### Збірка проекту для оптимізації
```shell
npm run build-dev
```

### збірка проекту з оптимізацією
```shell
npm run build-prod
```

### Очистка папки dist
```shell
npm run clear
```
